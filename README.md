# User Website Functionality Template using Ruby on Rails

## How to use

1. Install [ruby](https://rubyinstaller.org/)
 
2. Install bundle
  ```shell
  gem install bundler
  ```
  
3. Install rails
  ```shell
  gem install rails
  ```
  
4. Run the database migration
  ```shell
  rake db:migrate
  ```
  
5. Run the server locally
  ```shell
  rails s
  ```

## Functionality
 * Sign up
 * Login - Logout
 * Change username - password
 * Delete account
